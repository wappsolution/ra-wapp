# RA WAPP — Sistema de Gerenciamento com Realidade Aumentada

**RA WAPP** é um sistema web voltado para a gestão de equipamentos, conteúdos de treinamento e experiências com realidade aumentada (AR). Seu objetivo é centralizar recursos visuais interativos, facilitando o acesso a vídeos, QR Codes e estatísticas de uso.

---

## 🚀 Funcionalidades

- 📦 Cadastro e controle de **equipamentos**
- 🎥 Upload e gerenciamento de **vídeos de treinamento**
- 📱 Geração e leitura de **QR Codes para AR**
- 📊 Visualização de **indicadores e estatísticas**
- 🔐 Autenticação com login seguro

---

## 🧱 Tecnologias

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP (WAMP)
- **Banco de Dados**: MySQL
- **Ambiente local**: WAMP64 (Windows Apache MySQL PHP)
